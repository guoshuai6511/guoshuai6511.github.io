win+r 原生快速启动应用

发表于 2018-02-01 | 分类于 善用佳软 |

简介

Win + R，应该叫做windows run，是 windows 系统下用来快速启动应用程序的一个软件，
市面上固然有很多不错的快速应用启动应用，但能用原生实现的，并且速度还最快又何乐而不为呢。

这种操作并不高深，简单说就是给软件的快捷方式配置下系统路径。系统自带的程序，我们可以直接在 Win + R 的
输入框里输入，并按下 Enter 即可。对于非系统自带我们也只需要简单设置下即可，以后就操作飞快，还有机会在同事面前小秀一把。

自定义需要快速启动的软件

你可以指定添加某个应用程序的快捷方式用于快捷启动，但通常我们会给常用的一些软件都添加进来，所以我们需要建一个文件夹用来放这些快捷方式。
文件夹放到哪个位置都行，比如命名为 myshortcuts。

配置path

桌面 -> 我的电脑 -> 右键 -> 属性 -> 高级 -> 环境变量 -> 系统变量 -> path（选中该行，点击下方编辑按钮，在后面添加上你刚才新建文件夹的路径）。
